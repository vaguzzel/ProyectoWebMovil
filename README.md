### Cómo ejecutar el proyecto
Debes tener el node.js instalado

1. Clona este repositorio o descarga el `.zip`.
2. En la raíz del proyecto, ejecuta:

```bash
npm install
ionic serve
```

3. Se abrirá la app funcionando en tu navegador.
